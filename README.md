# Discord-js User Status Bot

## Installation
**Needed:**   
- Git (https://git-scm.com/download/)
- Node.js (https://nodejs.org/en/)

**Download:**   
Run the `git clone https://github.com/noelvissers/djs-userstatus.git`   command where you want the project.   
   
**Initialize:**   
- open the `cmd.bat` in the git repo.
- initialize npm by running the `npm install` command.
- start the bot by running `npm start`
- done.
